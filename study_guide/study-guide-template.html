AI Generation Template - Mẫu tạo tài liệu bằng AI

Study Guide HTML Template

Template này dùng để AI tạo tài liệu học tập và ôn phỏng vấn từ nhiều loại đầu vào: bài học lý thuyết, code, repo dự án, transcript, slide, ghi chú, hoặc kết quả brainstorming. Giữ nội dung vừa đủ với lượng input: ngắn khi input ít, sâu hơn khi là repo hoặc project lớn.

Áp dụng cho - Use cases RAG, Agentic AI, Deploy AI, Coding Agent, Docker, code review, project review, interview prep.
Quy tắc thuật ngữ - Terminology rule Với cụm technical term, viết dạng English - nghĩa tiếng Việt. Với từ đơn phổ biến, viết dạng cost (chi phí). Ví dụ: Semantic retrieval - truy xuất theo ngữ nghĩa, exact keyword match - khớp chính xác từ khóa.

Cách dùng nhanh - Quick usage

Đọc template, đọc input người học cung cấp, hỏi thêm bằng brainstorming nếu còn mơ hồ, sau đó tạo file HTML hoàn chỉnh. Không để placeholder trong bản cuối.
